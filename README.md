## ❓ TodoMonster
<div align="center">
<img src="https://user-images.githubusercontent.com/72336796/220352380-04df8bac-0d70-466b-a6fa-fb6bc298cdf8.png" height="500px" width="226px"></img>
</div>
<br />

**Todo Monster**는 몬스터 육성 + 유저들과 Todo list 공유가 가능한 To Do 앱입니다 !
- 재미요소를 위해 몬스터 육성 요소를 추가하였어요.
- 다른 사람의 Todo list를 구경하고, 팔로우도 할 수 있어요.
- Todo list를 카테고리별로 관리할 수 있어요.

## 📚 Tech
- Html, css, javascript, React.js
- Node.js
- MySQL

## 🌎 Setting
- Visual Code
- MySQL Workbench

## 🖐 담당 업무
> 회원 관리, 카테고리 관리 기능을 담당하였어요.
1. 메인페이지
2. 회원가입
3. JWT토큰 이용 로그인, 로그아웃
4. 비밀번호 변경
5. 프로필 변경
6. 계정 삭제
7. 카테고리 CRUD
8. 앱 대표 이미지 디자인

## 🤔 How To Use

- 프로젝트 실행
```
npm install
```

```
npm start
```
- Node 서버 실행
```
cd server
```
```
npm install
```
```
node server.js
```
